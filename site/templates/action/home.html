Basic Directory Structure

Main directories are:

  • cache

    Every dynamic content ends up here, things like pre-compiled PHPs, temporary thumbnails, etc.

  • config

    Main system configurations and definitions.

  • includes
  • libraries

    Here you can drop every library you project may require.

  • modules
  • shell
  • site

    Site's customized controllers and views.

    • controllers
    • langs
    • models

      Press this link to use example model.

    • scripts
    • styles
    • templates
